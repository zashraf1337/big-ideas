# Big Ideas — Tech Breakthroughs Tracker

A weekly-updated tracker of 15+ major tech breakthroughs (2021–2026) across AI,
bio, energy, space, quantum, robotics, and hardware.

**Live dashboard:** https://zashraf1337.github.io/big-ideas/

The dashboard, structured data (`state.json`), and dated snapshots (PDF / DOCX /
XLSX) are regenerated every Friday by an autonomous Claude research task. Each
weekly run scans primary sources for the last 7 days, merges new findings into
the base truth, and republishes here.

See `PROJECT.md` for the full architecture and protocol.
